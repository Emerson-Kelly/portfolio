CookieJar

Secure cloud-based file uploader with role-based access.

CookieJar is a full-stack file management system that allows users to upload, rename, delete, and organize files within authenticated accounts. It includes protected routes, session-based authentication with Passport.js, Prisma ORM integration, and cloud storage support.
